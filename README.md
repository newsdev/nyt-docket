## Getting started
```
pip install nyt-docket
```

## Using nyt-docket
### Demo app
Run the demo app.
```
python -m docket.demo
```

### Modules
Use the docket loader manually from within your Python script.

#### Grants (new cases)
```python
from docket import grants

g = grants.Load()
g.scrape()

for case in g.cases:
    print case.__dict__
```

#### Slip opinions (decisions)
```python
from docket import slipopinions

o = slipopinions.Load()
o.scrape()

for case in o.cases:
    print case.__dict__
```
